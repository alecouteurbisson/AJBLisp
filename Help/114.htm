(equal <this> <that>)

Function

Compares any two objects for equality of type and value. The equality of lists and vectors is determined element by element.



Examples: 

> (equal  '(a b c) '(a b c))
= t                    ; Equal (but not eq) lists

> (equal 1 1)
= t                    ; Equal integers

> (equal 1 1.0)
= nil                  ; Different types
