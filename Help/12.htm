(rplacd <list> <value>)

Function

Replace the cdr of <list> with <value>.

This is a destructive operation and will affect all references to the list. It is possible to set the cdr of a list or sub-list to the list itself. This is OK until the resulting list is scanned (e.g. by print.) An endless loop will result which can be broken by pressing the [Esc] key.



Examples: 

> (setq x '(1 2 3) y x)   ; x = y = (1 2 3)
= (1 2 3)

> (rplacd x 'z)           ; Modify x
= (1 . z)

> y
= (1 . z)                 ; Y affected similarly
