(typep <any> <type>)

Function

Predicate; true if the first arg has the type specified by the second. If the second arg is not a valid type key then the result is always nil.

Valid type keys are: 

nil
:cons       :symbol
:constant   :key
:character  :string
:integer    :float
:vector
:subr       :subrl
:fsubr      :fsubrl
:file       :window
