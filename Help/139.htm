(lexcmp <x> <y>)

Function

Compare <x> and <y> in a consistent way. Types are ordered in an arbitrary way and within each type the values are ordered arbitrarily (although, if a meaningful order exists then it is used.) The result is -1, 0 or 1 for <x> < <y>, <x> = <y> or <x> > <y>. This function is used to impose a canonical order on lists and vectors so that they may be more easily compared and analysed.
