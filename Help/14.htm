(apply <exec> <args>)

Function

Apply <exec> to a list of arguments, <args>. The argument list is not evaluated further even if <exec> is a function, rather than a macro.



Examples: 

> (apply + '(1 2 3))
= 6             ; Value of (+ 1 2 3)

> (apply car '((2 3)))
= 2             ; Value of (car '(2 3))
