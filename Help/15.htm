(typeof <any>)

Function

Return the type of any lisp object.

The type will always be one of the following, 

 nil
 :cons       :symbol
 :constant   :key
 :character  :string
 :integer    :float
 :vector     :cvector
 :subr       :subrl
 :fsubr      :fsubrl
 :file       :window



Examples: 

> (mapc typeof '(a 1 3.0))
= (:symbol :integer :float)
