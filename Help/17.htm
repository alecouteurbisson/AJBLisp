(symbol <name>)

Function

Return a symbol with the given <name> string. The symbol is created if it doen not already exist. (symbol "nil") will return nil even though nil is not a symbol since this behaviour is useful and probably expected.



Examples: 

> (symbol "t")        ; Just returned
= t

> (symbol "f*&==@##")
= *&==@##!            ; Probably created   :)
