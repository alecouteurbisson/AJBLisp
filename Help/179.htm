(and [<boolean> ...])

Macro

Evaluates its arguments from left to right until one of them is nil whereupon it returns nil. If all arguments are not nil or if there are no arguments then it returns t.

In addition to its use as a boolean operator it can also be used to check pre-conditions (although when/unless are better suited for this purpose.)



Examples: 

> (and)                  ; Degenerate case
= t

> (and t nil)
= nil

> (and (consp x) (pop x))
= nil                     ; x was not a cons so (pop x)
                          ; was never evaluated
