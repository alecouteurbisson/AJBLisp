(or [<boolean> ...])

Macro

Evaluates its arguments from left to right until one of them is not nil whereupon it returns t. If all arguments are nil or if there are no arguments then it returns nil.

In addition to its use as a boolean operator it can also be used to check pre-conditions (although when/unless are better suited for this purpose.).



Examples: 

> (or)                 ; Degenerate case
= nil

> (or t nil)
= nil

> (or x (do_it))       ; Only do_it if x is NIL
= t
