(if <pred> <then> [<else>])

Macro

If <pred> evaluates to nil then evaluate and return <else> otherwise evaluate and return <then>.

If <else> is omitted and <pred> evaluates to nil then return nil.



Examples: 

> (if (= 3 3) 'Equal 'Unequal)
= Equal

> (if (= x 3) "X is 3" "X is not 3")
= "X is not 3"
