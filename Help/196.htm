(for-each (<var> <vect-or-list> [<index>] [<retn>]) <body>...)

Macro

Evaluate the expressions in <body>... with symbol <var> taking its values from the top-level elements of <vect-or-list> <var> is local to the loop. If the symbolic argument <index> is present then it also becomes a local variable that counts from 0 on each loop. for-each returns the value of the expression <retn> or the value of the last body expression evaluated if omitted. <index> may be null if it is not required but a return expression is.

Both <var> and <index> are valid <retn> values. break, while and until may be used to end the loop prematurely.



Examples: 

> (for-each (k #(a b c d)) (print k " "))
a b c d = nil

> (for-each (k '(a b c 1 2 3)) (until (numberp k) (string k)))
= "1"

> (for-each (k '(a b c 1 2 3) i i) (when (numberp k) (println i :space k)))
3 1
4 2
5 3
= 5
