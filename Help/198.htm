(until <cond> [<body> ...])

Macro

If <cond> is non-nil then evaluate the expressions in <body> in turn, returning the last. If <body>... is omitted then return the value of <cond>. In addition, the next time that control returns to the top level of a loop function then the loop will exit with the value of the top-level expression that halted it as the result.

It is intended that until only be used at the top- level of a loop and in this case the operation is obvious.

For/foreach loops will also exit but the return value is discarded.
