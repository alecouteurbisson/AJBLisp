(quote <literal>) or '<literal>

Macro

Protects its argument from evaluation.

'x is a shorthand for (quote x) and is converted to the latter form immediately upon input.



Examples: 

> ''a
= (quote a)            ; One quote evaluated away

> (setq a '(x y z))    ; Set a to a literal list
= (x y z)

> (atom a)
= nil                  ; The value of a is not an atom

> (atom 'a)
= T                    ; ...but a (the symbol) is itself an atom
