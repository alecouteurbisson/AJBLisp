(break <body> ...)

Function

Evaluates its arguments like progn, returning the last. In addition, the next time that control returns to the top level of a loop function then the loop will exit with the entire value of the expression that halted it as the result.

For/foreach loops will also exit but the return value is discarded

Equivalent to (while nil <body>...) inside a loop.

Note that a for/foreach loop will discard the return value of break.
