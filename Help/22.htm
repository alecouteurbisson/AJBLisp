(remprop <symbol> [<name>])

Function

Remove the named property from the symbol. If <name> is omitted then remove all properties and return the original property list. Otherwise returns the value of the property removed or :undefined if the named property was not present.



Examples: 

> (remprop 'apple 'colour)
= "green"                ; for example

> (remprop 'apple 'colour)
= :undefined
