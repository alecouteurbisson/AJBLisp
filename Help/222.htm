(cd [<string>])

Function

Change the current directory and drive. This is the only way to change the current directory. (system "cd <path>") will not work as it only changes the directory for the newly created process.

Beware that (cd "c:\") and the like will apparently lock up the listener because the correct command is (cd "c:\\") or (cd "c:/"). If you get stuck like this then type ") to close the string and the list and re-type the command correctly. cd returns nil, unless no argument is given, where it returns the current directory as a string.



Examples: 

> (cd)
= "E:\Work\Prog\CPP\AJBLisp\Distribution"

> (cd "..")
= nil

> (cd)
= "E:\Work\Prog\CPP\AJBLisp"
