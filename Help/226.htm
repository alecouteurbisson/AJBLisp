(environ [<var> [<value>]])

Function

Read/write environment variables. With no arguments it returns the whole environment as a list of strings.

With one string argument it returns the value of the named variable as a string or nil if not found. With two string arguments it sets <var> to <value> and returns a boolean which is true if successful.



Examples: 

> (environ "windir")
= "C:\WINDOWS"
