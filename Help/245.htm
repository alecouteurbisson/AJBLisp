(gpen <window> <colour> <width/style>)

Function

Set the line drawing style for a window. The colour is an integer colour. The width/style argument selects the line width if >= 1 and sets the line style if <= 0. Note that styled lines are only available with a width of 1.

The :ps_* constants are the available pen line styles: 

 :ps_solid,   :ps_dash,       :ps_dot,
 :ps_dashdot, :ps_dashdotdot, :ps_clear
