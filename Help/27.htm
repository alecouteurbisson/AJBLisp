(let ((<symbol> <value>) ...) <body> ...)

Macro

Creates local variables. The expressions in body are evaluated in turn and the value of the last is returned. The first argument is a list of variable bindings that exist only during the evaluation of <body> ...

Each variable binding is itself a list of two items, a symbol and an expression that is evaluated and initially bound to the symbol. All of the <value> expressions are evaluated in the environment surrounding the let.



Examples: 

> (let ((a 1) (b 2))
       (+ a b) )
= 3
