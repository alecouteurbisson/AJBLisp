(length <list>), (length <vector>), (length <string>)

Function

Return the length of a vector, list or string. It counts only the top level elements; elements which are themselves vectors or lists are counted once. For a string argument it returns the length of the string in characters.



Examples: 

> (length '())
= 0

> (length '(a b c))
= 3

> (length '(a b . c))
= 3

> (length #(1 2 3 4 5))
= 5

> (length "Hello")
= 5
