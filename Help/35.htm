(append [<list>]...)

Function

Create a single list containing the top-level items of each list concatenated. The rightmost argument may be an improper (dotted) list. The other arguments must be proper lists.



Examples: 

> (append)
= nil

> (append '(a b c) '(1 2 3))
= (a b c 1 2 3)

> (append '(a b c) '(1 2 . 3))
= (a b c 1 2 . 3)

> (append '(a (b c)) '((1 2) 3))
= (a (b c) (1 2) 3)

> (append '(a b c) '(1 2 3 4) '(+ - * /))
= (a b c 1 2 3 4 + - * /)
