(reverse <list>) (reverse <vector> [<len>])

Function

Returns a list of the top level items of <list> reversed.

With a vector argument it returns the first <len> items reversed into a new vector of size <len>. <len> defaults to the length of the input vector.



Examples: 

> (string (reverse (chars "Madam I'm Adam")))
= "madA m'I madaM"
