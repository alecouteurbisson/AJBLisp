(member <atom> <list>)

Function

Return a true value if <atom> is a top-level member of <list> else return nil. The true value returned is the sublist of <list> starting with <atom> rather than the symbol t.



Examples: 

> (member #\p (chars "open"))
= (#\p #\e #\n)

> (member #\q (chars "open"))
= nil
