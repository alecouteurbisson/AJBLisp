(filter <pred> <list> [<keep>])

Function

Returns <list> with its top-level items filtered. Each item is passed to function <pred> and those which produce a result equal to the boolean <keep> are kept. <keep> is t by default, (i.e. the predicate must return true to keep an item.)



Examples: 

> (filter numberp '(a 1 b c 3 5 d))
= (1 3 5)

> (filter numberp '(a 1 b c 3 5 d) nil)
= (a b c d)
