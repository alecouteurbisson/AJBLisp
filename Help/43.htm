(push <item> <sym>)

Macro

Push any item onto the front of the list. <sym> must be a symbol and its value will be modified. The value of sym need not be a list and the result will be a dotted pair in this case.

Returns the modified list. The item can be removed with pop.



Examples: 

> (setq stack nil)
= nil

> (push stack 'a)
= (a)

> (push stack 'b)
= (b a)

> (pop stack)
= b

> (pop stack)
= a

> stack
= nil
