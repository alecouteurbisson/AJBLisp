(vsort <vector> [<cmp>]>

Function

Sort a vector. (<cmp> a b) should return an integer, <0 if a < b, 0 if a = b and >0 if a > b assuming an ascending sort. Negate these results for a descending sort (or reverse the result.) The default value for <cmp> is lexcmp which does the right thing for ascending sorts of homogeneous vectors and canonicalisation of inhomogeneous vectors.
