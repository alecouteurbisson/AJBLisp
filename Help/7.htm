(psetq <symbol> <value> [<symbol> <value>] ...)

Macro

Sets the <symbol>s to the <value>s in parallel so that all values are evaluated in the enclosing scope. The symbol arguments are not evaluated and should not be quoted. Returns the last value bound.



Examples: 

> (setq x 1 y 2)
= 2

> (psetq x y y x)    ; Value of y set to previous value of x
= 1
(println x :space y)
2 1                  ; Values swapped
= 1
