(string-search <string> <string>) (string-search <string> <char>)

Function

Searches the first argument string for the first ocurrence of any of the characters in the second argument. If found, then the matching character's index is returned, otherwise return nil.



Examples: 

> (string-search "1,2,3" #\,)
= 1

> (string-search "abc efg;mno" "; ")
= 3
