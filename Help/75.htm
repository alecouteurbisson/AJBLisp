(tzset [<zone>])

Function

Sets the timezone according to the TZ environment variable. If a timezone string is provided then TZ will be set to this value before setting the timezone itself. Therefore (tzset <string>) is identical in effect to (environ "TZ" <string>) (tzset).

The TZ variable can be read by evaluating (environ "TZ") but the returned value is not necessarily the current timezone unless tzset has been evaluated with or without an argument. To avoid confusion it is best to explicitly set the timezone in init.lsp.

Neither the argument or the value of TZ are checked for validity. If an argument is provided it will be truncated after the 16th character.



Examples: 

> (tzset "GMT0BST")
