(open-file <filename> [<mode> [<text>]])

Function

Opens the named file with the specified mode. <mode> may be :read (readonly, the default), :write (read/write) or :create (create, read and write).

If <text> is present and true then open as a text file. This function is the only way to create a File atom.



Examples: 

> (setq f (open-file "c:\\temp\\scratch" :create))
= #<File:0x0004d6c7>
