(float-format [<string>])

Function

Read or set the format for floating output. The format string should be a C language format as used by printf. This format is used by string, print and println. The formats used by write and writeln are fixed to provide re-readable input with full accuracy. Remember that Lisp floats are C doubles and the format should be of type lg, le, lE or lf. print-format is a more flexible (and modeless) option.



Examples: 

> (float-format "%10.4lf")
= "%10.4lf"

> (println (sqrt 3))
      1.7321
= 1.732050807568877
